TOR: What is Tor and should I use it?

Out of many the biggest one is the same that you have used to ask this Question i.e. Anonymity over the Internet. Also it is the major tool to access the Deep Web, the sites which are not indexed by big search engines like Google,Yahoo,etc only because they are not allowed to. But you can access those using Tor. Some Facts about Deep Web:-

  • It contains about 500 times more data than the surface web which we never access.
  • We have access to only 0.03 % of data on the internet, which we call surface web.
  • The deep Web contains 7,500 terabytes of information compared to nineteen terabytes of information in the surface Web.
